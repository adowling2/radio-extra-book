\chapter*{How to Use This Book}
\addcontentsline{toc}{chapter}{How to Use This Book}
Start with the cheat sheet if you are close to the exam and need a compact
formula review. Then read the explanatory chapters in order if you want the
engineering story behind the formulas. The recurring pattern is:
\begin{enumerate}
\item identify what is conserved;
\item choose the natural energy-storage variables;
\item write the differential equation or state model;
\item convert to phasors for sinusoidal steady state;
\item interpret the same result as energy storage, dissipation, gain, phase, or
impedance.
\end{enumerate}

The sidebars separate exam-level facts from controls and energy interpretations.
For quick review, read the \emph{Exam Relevance}, \emph{Common Mistake}, and
\emph{Key Equations} sections first.
